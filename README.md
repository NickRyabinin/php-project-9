### Page Analyzer – сайт, который анализирует указанные страницы на SEO пригодность.

### Требования:
php >= 8

composer

### Установка:
```bash
git clone git@github.com:NickRyabinin/php-project-9.git

make install
```
### Локальный запуск:
```bash
make start-local
```
### Запуск в продакшене:
```bash
make start
```
